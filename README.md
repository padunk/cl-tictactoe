# TIC TAC TOE

## Installation

```bash
npm i
```

## Run on local server

```bash
npm run dev
```

Then open a new tab in your web browser and go to `http://localhost:5173`

## Run on preview

```bash
npm run build
npm run preview
```

Then open a new tab in your web browser and go to `http://localhost:4173`

## Running test

```bash
npm run test
```
